OCR+
OCR+ é um serviço online de OCR(Optical Character Recognition), utilizado para extrair o texto de imagems.

Como ele funciona?

A ferramenta permite extrair texto de imagens utilizando o Tesseract, um software de reconhecimento ótico de caracteres.
Além disso, caso desejado, utilizando o Puppeteer, também é possível traduzir o texto extraído.
